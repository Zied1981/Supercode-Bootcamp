Anlegen & wechseln

git branch:zeigt alle Branches die vorhanden sind

*:zeigt in welchem branch man sich befindet

git branch:legt eine neue branch an

git checkout: wechselt zu einem neuen branch

git checkout -b :legt eine neue branch an und wechselt direkt dahin.

git checkout -d :löscht den branch

git Merge: versucht den main branch mit dem side branch zusammenführen
